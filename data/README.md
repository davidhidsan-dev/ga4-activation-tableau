# data/

## ES

Esta carpeta contiene los **CSVs exportados desde BigQuery** que se usan para construir el dashboard en Tableau (Tableau Public).
Son **tablas derivadas/agregadas** (no datos raw a nivel evento).
La fuente de estos exports son las consultas en `/sql`.

## EN

This folder contains **CSV exports from BigQuery** used to build the Tableau dashboard (Tableau Public).
These are **derived/aggregated tables** (not raw event-level data).
The source of these exports is the SQL queries in `/sql`.
